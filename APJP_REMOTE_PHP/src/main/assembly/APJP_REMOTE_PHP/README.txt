PRE-INSTALLATION
----------------
- install PHP >= 5.0.0 + MCRYPT >= 2.5.6 + OPENSSL >= 0.9.6
- install PHP SERVER

INSTALLATION
------------
- unzip APJP_REMOTE_PHP-X.X.X.zip
- open APJP_REMOTE_PHP-X.X.X
- open index.php
	- APJP_APJP::$APJP_KEY == generated key
	- APJP_APJP::$APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_KEY[X]
	- APJP_APJP::$APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_VALUE[X]
	- APJP_APJP::$APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_KEY[X]
	- APJP_APJP::$APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_VALUE[X]
- install

IMPORTANT
---------
- APJP_REMOTE_HTTP_SERVER_X_REQUEST_URL == http://../?HTTP
- APJP_REMOTE_HTTPS_SERVER_X_REQUEST_URL == http://../?HTTPS
